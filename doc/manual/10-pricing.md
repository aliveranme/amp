# 定价与商业版

> 来源：[https://ampcode.com/manual](https://ampcode.com/manual)

## 定价模式

- 按实际 LLM 使用量 + 部分工具（如网页搜索）收费
- 个人和非企业工作区**无加价**
- 无订阅或承诺，最低购买 $5 美元信用额度
- 示例：$2.50 从信用额度中扣除（$2 给 Anthropic + $0.50 给 OpenAI）
- 在用户设置、工作区设置或 `amp usage` 中购买信用额度和查看余额

## Enterprise 版本

Enterprise 比个人/团队贵约 50%。包含：

| 功能 | 说明 |
|------|------|
| SSO | Okta、SAML 等 |
| 目录同步 | — |
| 零数据保留 | LLM 推理的文本输入不留存 |
| 线程可见性控制 | 高级控制 |
| 用户成本控制 | 按用户额度管理 |
| MCP 注册表允许列表 | — |
| 托管用户设置 | 管理员强制执行 |
| API | 工作区分析/数据管理（OpenAPI schema at `/api/v2/openapi.json`） |
| 用户组 | 成本归属和按组可见性（按需） |
| 线程保留期 | 可配置（按需，额外收费） |
| IP 白名单 | 工作区访问控制（按需，额外收费） |
| BYOK 模型提供商 | 支持区域端点 |
| 一次性费用 | $1,000 USD 开启 Enterprise |

## 其他说明

- 点击线程侧边栏的 `$` 查看详细成本信息
- 未使用的信用额度在账户不活跃一年后过期
- 工作区信用额度汇总共享
- 发票通过 Stripe 处理（支持 VAT ID）
